Methors

  • What is the length property in a string?
    It returns the total number of characters in a string.
  • Difference between slice() and substring()?
    slice() supports negative indexes; substring() does not.
  • What does trim() do?
    Removes leading and trailing whitespace.
  • How does replace() differ from replaceAll()?
    replace() changes only the first occurrence, while replaceAll() changes all occurrences.
  • What does split() return?
    It converts a string into an array using the given separator.
  • What is toUpperCase() used for?
    Converts all characters in a string to uppercase.
  • How does includes() work?
    Checks if a substring exists inside a string and returns true or false.
  • What is parseInt() used for?
    Converts a string to an integer, stopping when a non-digit is found.
  • Difference between parseFloat() and Number()?
    parseFloat() reads decimal values even with text; Number() requires a fully valid number.
  • What does toFixed() do?
    Formats a number to a fixed number of decimal places.
  • What is toPrecision() used for?
    Formats a number to a specific number of significant digits.
  • What is isNaN() used for?
    Checks if the given value is Not-a-Number.
  • What is Math.random()?
    Generates a random number between 0 and 1.
  • Difference between Math.floor() and Math.ceil()?
    Math.floor() rounds down; Math.ceil() rounds up.
  • What does Math.round() do?
    Rounds a number to the nearest integer.
  • What does Math.max() and Math.min() return?
    They return the highest and lowest values among the given numbers.
  • What does Math.abs() do?
    Returns the absolute (positive) value of a number.
  • What is Math.pow(a, b)?
    Calculates a raised to the power b.
  • Why do we use number conversion methods?
    To ensure values are treated as numbers for accurate calculations.
  • Why are string methods important?
    They help manipulate, clean, and process text efficiently.
