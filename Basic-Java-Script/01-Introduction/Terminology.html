1. Browser

    2. A browser (also called web browser) is a software/application that you use to open and view websites on the internet.

  2. Console

    3. The browser console is a developer tool that shows errors, warnings, and messages from a website. It also allows you to run JavaScript code directly and debug webpages. Developers use it to check why something is not working on a site.

  3. Script Tag

    4. Use to write code of java script in html file or to link external file

  4. Syntax

    5. Syntax means the set of rules that define how code must be written in a programming language.

  5. Statement

    6. A statement is a single instruction in a program that tells the computer to do something.

  6. Variable

    7. A variable is a container used to store data or values in a program.

  7. Runtime

    8. Runtime means the period when a program is actually running or executing.

  8. Compiler

    9. A compiler translates the entire program into machine code before running it.

  9. Interpreter

    10. An interpreter reads and executes the code line by line while the program is running.

  10. Error Message

    11. An error message is a warning shown by the computer or browser when something is wrong in your code. It tells you what the problem is so you can fix it.

  11. Developer Tools

    12. Developer Tools are built-in tools in a browser that help developers inspect, debug, and test websites. They include Elements, Console, Network, Sources, and Performance tabs to check code, errors, styles, and webpage behavior.
